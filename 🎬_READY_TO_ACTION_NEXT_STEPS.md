## 🆘 IF SOMETHING BREAKS

**Super Quick Fixes:**

| Error | Fix Command |
|-------|------------|
| Port 5000 in use | `taskkill /F /IM node.exe` |
| Port 5175 in use | Frontend auto-retries 5173-5176 |
| Module not found | `npm install` (in that folder) |
| Won't start | `npm cache clean --force && npm install` |
| Page blank | Press F5 to refresh |
| 400 error | Restart backend (Ctrl+C, npm run dev) |

**For detailed help:** Open 🆘_TROUBLESHOOTING_GUIDE.md
